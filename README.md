# React Task Management Dashboard
## Project Description

The React Task Management Dashboard is a responsive web application developed using React. It allows users to create, manage, update, search, filter, and track tasks efficiently through a clean dashboard interface.

This project demonstrates React fundamentals including Components, JSX, useState, useEffect, Context API, React Router, Local Storage, and CRUD operations.

---

## Features
 
### Authentication

* Login Functionality
* Logout Functionality
* Protected Dashboard Route

### Dashboard

* Dashboard Overview
* Total Tasks Count
* Completed Tasks Count
* Pending Tasks Count

### Task Management

* Create Task
* Update Task
* Delete Task
* Change Task Status
* Search Tasks
* Filter Tasks by Status

### Data Persistence

* Local Storage Integration
* Task Data Retained After Refresh

### State Management

* Context API
* React Hooks (useState, useEffect, useContext)

### User Interface

* Responsive Bootstrap UI
* Dashboard Cards
* Status Badges
* Confirmation Dialogs

## Installation Steps

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Project Folder

```bash
cd task-dashboard
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Open Browser

```text
http://localhost:5173
```
